最近工作太努力了, 小拇指 RSI 又间歇性发作, 很是头疼.

逛 google 突然发现一个项目叫做 spacemacs, 竟然有6000多个 star.
拿过来用了用, 使用下来, 感觉比我维护了那么多年的 dot emacs 要好很多, 虽然还差一些插件, 但是有很多地方配置的比我自己配置的要好很多.

这篇文章介绍的不错, 可以先看看: http://hackerxu.com/2015/08/31/spacemacs.html

我这里主要补充几个地方, 这个 spacemacs 有三种模式:
1. evil mode, 主要是 vi 为主的 mode. 也就是 control 模式和编辑模式通过插入来切换的. 这时候主键是 space 键.
2. holy mode, 主要是以 emacs 模式, 主要以 emacs 的方式为主, 要进入 spacemacs 里面, 就使用 M-m为主. 但是移动和选择那些按键还是 emacs 的.
3. hybrid mode, 在 edit mode 下面, 和 emacs 一样, 通过 ESC 来切换到 vi 模式. 一种混合模式.
