色戒是部好电影

发表于 2008-01-13 | 分类于 未分类

《色戒》
一部难得的好电影。

等待， 等待， 终于下好了完整版。
等待， 等待， 终于一起看完了他。

初衷呢， 很庸俗， 为了剪掉的那部分。
以为会很轻松。看完之后却很沉重。让我看到了抗战时候沦陷区的生活， 战争的对那些还活着的人的折磨， 想象一下我们的生活在他们看来有多幸福吧。

有人说这是一部汉奸题材的电影。还不如说， 这是一部爱国主义电影， 让我知道了我们的生活得来不易， 那些祖辈作出的牺牲是我们不可想象的， 似乎我们已经对那些战场上牺牲的画面习以为常， 也许那只是最不煎熬的一部分吧。 在他们的后面， 在那些沦陷的城市中， 受过的欺凌还有很多很多。 惭愧， 请我们亲爱的媒体们不要再宣扬什么小米加步枪能打得过机枪加大炮了， 让我们有一种胜利理所应当的感觉， 也许我们真的有5个壮士很勇敢守了一个悬崖很多天的故事把， 但是那只是少数的少数， 而且这种宣扬， 可能是战争那种特殊时期鼓舞人心的办法吧。现在， 请我们都诚实的面对，我们和他们的差距真的很大。

色戒是一部关于性和爱和女性的电影， 不讲爱国， 单拿出来王佳芝自己来说， 让我们来体会一些她的感受， 母亲去世， 父亲再婚， 暗恋着好友的男朋友， 暧昧却不敢道明， 处子之身因为一场失误而献给了一个猥亵男。 一个始终被利用的角色， 也许只有易先生对她动了真情， 也许很多人认为易先生那不是爱吧， 但我觉得易先生在听完王佳芝的小曲以后， 是真正的爱上了这个女子，在那之前， 这个女子已经爱上了他。 在那样的环境下， 真爱难能可贵。可是谁让王佳芝是个女特务， 为了她的职责， 还是把易先生引到了暗杀地点， 如果在“上级”能够不把她当成一个棋子， 也许他们会成功的， 可是她确实就是“上级”的一个棋子， 她的诉求不会得到任何形式的尊重， 就连她给她父亲的信也被“上级”烧掉了， 所以我很同情她， 她确实做了一个女人该做的事情。女人对爱的回报让我感到惭愧。

再说说爱吧， 感谢李安拍出这么好的一部电影， 让我懂了爱的伟大。 如果你没有感觉到，请用心体会。

最后，色戒是部好电影，大家都去看吧， 不要跳着看哦～

动态库编译方法[转]

发表于 2007-11-23 | 分类于 未分类

# 声称动代连接库，假设名称为libtest.so
gcc x.c y.c z.c -fPIC -shared -o libtest.so

# 将main.c和动态连接库进行连接生成可执行文件
gcc main.c -L. -ltest -o main

# 输出LD_LIBRARY_PATH环境变量，一边动态库装载器能够找到需要的动态库
export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:.

# 测试是否动态连接，如果列出libtest.so，那么应该是连接正常了
ldd main

# 执行就不用说了吧

--------------------------------------------------
建议创建一个Makefile来作这些事情。

-fPIC：表示编译为位置独立的代码，不用此选项的话编译后的代码是位置相关的所以动态载入时是通过代码拷贝的方式来满足不同进程的需要，而不能达到真正代码段共享的目的。

-L.：表示要连接的库在当前目录中

-ltest：编译器查找动态连接库时有隐含的命名规则，即在给出的名字前面加上lib，后面加上.so来确定库的名称

LD_LIBRARY_PATH：这个环境变量指示动态连接器可以装载动态库的路径。
当然如果有root权限的话，可以修改/etc/ld.so.conf文件，然后调用
/sbin/ldconfig来达到同样的目的，不过如果没有root权限，那么只能采用输出LD_LIBRARY_PATH的方法了。
